# Slides for EUMAS 2023 main conference 

### Download
```bash
git clone --recursive git@github.com:anitvam/slides-eumas-2023.git
```

### Update
```bash
git pull --resurse-submodules
```

### Build
```bash
hugo
```

### Local serve
```
hugo server
```
available at `http://localhost:1313/slides-eumas-2023/#/`.
